# 新UI设计说明

## 设计概述

基于 `docs/ui.png` 设计图，重新设计了应用界面，采用现代化的渐变背景和玻璃态设计风格。

## 主要变更

### 1. 视觉设计

#### 背景
- **渐变背景**: 从紫色到粉色的渐变效果
  - 起始色: `#667eea` (紫蓝色)
  - 中间色: `#764ba2` (紫色)
  - 结束色: `#f093fb` (粉色)
- **固定背景层**: 使用独立的 `.gradient-bg` 层实现

#### 玻璃态效果 (Glassmorphism)
- **状态卡片**: 半透明黑色背景 + 模糊效果
  - 背景: `rgba(0, 0, 0, 0.3)`
  - 模糊: `backdrop-filter: blur(10px)`
  - 圆角: `16px`
  - 阴影: `0 8px 32px rgba(0, 0, 0, 0.2)`

- **设置区域**: 更浅的半透明背景
  - 背景: `rgba(0, 0, 0, 0.2)`
  - 模糊: `backdrop-filter: blur(10px)`

### 2. 组件设计

#### 状态显示
- **状态指示器**:
  - 圆点 + 文本
  - 就绪: 绿色 (`#10b981`)
  - 录制中: 红色 (`#ef4444`) + 脉冲动画
  - 暂停: 橙色 (`#f59e0b`)

- **计时器**:
  - 字体大小: `48px`
  - 字重: `700` (粗体)
  - 颜色: 白色
  - 字间距: `2px`

#### 控制按钮
采用 2x2 网格布局，每个按钮使用渐变色：

1. **选择区域** (蓝色)
   - 渐变: `#3b82f6` → `#2563eb`
   - 图标: 网格图标

2. **开始** (绿色)
   - 渐变: `#10b981` → `#059669`
   - 图标: 圆形录制图标

3. **暂停** (橙色)
   - 渐变: `#f59e0b` → `#d97706`
   - 图标: 暂停图标

4. **停止** (红色)
   - 渐变: `#ef4444` → `#dc2626`
   - 图标: 方形停止图标

**按钮交互**:
- 悬停: 向上移动 2px + 增强阴影
- 点击: 恢复原位 + 减弱阴影
- 禁用: 50% 不透明度

#### 设置区域
- **下拉选择器**:
  - 半透明黑色背景
  - 白色文本
  - 自定义下拉箭头
  - 悬停/聚焦效果

### 3. 功能变更

#### 移除的功能
- ❌ 输出目录选择器（简化界面）
- ❌ 音频开关切换按钮

#### 新增的功能
- ✅ 音频源下拉选择
  - 关闭
  - 麦克风
  - 系统声音
  - 麦克风 + 系统声音

## 技术实现

### CSS 特性
- **CSS 变量**: 移除，改用固定颜色值
- **渐变**: `linear-gradient()` 用于背景和按钮
- **模糊效果**: `backdrop-filter: blur()`
- **过渡动画**: `transition` 用于交互效果
- **网格布局**: `display: grid` 用于按钮布局

### SVG 图标
使用内联 SVG 替代 emoji 图标，提供更好的视觉效果：
- 网格图标 (选择区域)
- 圆形图标 (开始)
- 暂停图标 (暂停)
- 方形图标 (停止)

### 响应式设计
- 固定窗口大小 (420x680)
- 自适应内容滚动
- 自定义滚动条样式
